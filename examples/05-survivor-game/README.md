# Survivor Game

React + SVG で作る Vampire Survivors 風のゲームです。

## 特徴

- 全方位から迫る敵を自動射撃で倒す
- 経験値を集めてレベルアップ
- ランダムな能力強化
- SVG によるスムーズなグラフィック

## 遊び方

1. プレイヤーは画面中央に固定されています
2. 敵は全方位から自動的に出現します
3. 最も近い敵に向けて自動的に弾が発射されます
4. 敵を倒すと経験値オーブがドロップします
5. 経験値オーブに近づくと自動的に回収されます
6. レベルアップするとランダムな能力が強化されます

## 強化要素

レベルアップ時にランダムで以下の能力が強化されます：

- **発射速度**: 弾の発射間隔が短くなる
- **弾速**: 弾の飛ぶ速度が上がる
- **弾の威力**: 敵に与えるダメージが増加
- **射程距離**: 弾が飛ぶ距離が伸びる
- **移動速度**: プレイヤーの移動速度が上がる（未実装）

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 技術スタック

- **React**: UI フレームワーク
- **TypeScript**: 型安全な開発
- **Vite**: 高速な開発環境
- **SVG**: ベクターグラフィックス

## ゲームの仕組み

### ゲームループ

`requestAnimationFrame` を使用して 60FPS でゲームを更新します。

### 衝突判定

円同士の距離計算による衝突判定を実装しています。

### カメラシステム

SVG の `viewBox` を使用してカメラの視点を制御しています。
